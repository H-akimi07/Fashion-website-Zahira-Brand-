# ZAHIRA - Luxury Afghan Heritage Brand Website

![ZAHIRA Website Preview](images/preview.png)

## ✨ Overview

ZAHIRA is a premium single-page luxury fashion website inspired by Afghan heritage, traditional craftsmanship, and modern elegance.

The project represents a digital experience for a luxury Afghan brand that showcases traditional clothing, handmade jewelry, and cultural products to a global audience.

The goal was to combine cultural identity with a modern luxury design approach.

---

## 🎨 Design Concept

The website was designed with a focus on:

- Luxury branding
- Minimal and elegant layouts
- Cultural storytelling
- Premium fashion aesthetics
- Smooth user experience

The visual identity uses a warm luxury palette inspired by:

- Gold
- Dark tones
- Earth colors
- Traditional craftsmanship

---

## 🚀 Features

- Responsive single-page design
